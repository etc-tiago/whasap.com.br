# @whasap/meta

Cliente HTTP para **WhatsApp Cloud API** (Meta Graph API).

## Documentação oficial da API

A collection Postman de referência está em [`doc-oficial.json`](./doc-oficial.json) neste pacote. Use-a para bodies de envio, templates, mídia e webhooks.

Implementação: [`src/client.ts`](./src/client.ts).

## Versão Graph API

Definida em `@whasap/config` → `mvpDefaults.meta.apiVersion` (atualmente `v21.0`).
